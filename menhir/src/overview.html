Description de package launcher

Ce package contient toutes les classes nécessaires à l'éxécution du code et plus précisemment au lancement du jeu du menhir. Il contient ainsi la classe Jeu avec la fonction main qui va, en fonction d'une constante, choisir de lancer le jeu du menhir en mode textuel avec la classe Console ou en mode graphique avec la classe GraphicLauncher.

Description de package modele

Ce package contient toutes les classes qui décrivent le modèle du jeu du menhir, c'est-à-dire les joueurs, les cartes et de manière plus générale les paquets de ressources de la partie ou des joueurs, le déroulement de la partie ou encore les stratégies de joueurs. Ce package contient ainsi toutes les classes qui vont permettre le bon fonctionnement du jeu au niveau du "coeur" de l'application car il exécute le jeu avec la classe Partie indifféremment du mode textuel ou graphique, c'est à dire que les classes du package s'occupent de tout ce qui ne concerne pas les entrée/sorties (à l'exception de la classe StrategieJoueurReelConsole qui s'occupe des entrées clavier directement pour le mode textuel) ou le lancement de l'application.

Description de package ressources

Ce package contient simplement les ressources graphiques du jeu pour que le package vue les affichent avec ses classes. Il contient aussi la classe Ressources qui va s'occuper de charger toutes les ressources de manière unique et centralisée pour que les classes du du package vue n'ai pas à les recharger, elles auront juste à en récupérer une référence. (les ressources sont libres de droit)

Description de package vue

Ce package contient toutes les classes permettant une interaction graphique avec l'utilisateur avec notamment la fenêtre d'initialisation du jeu, le fenêtre de gestion des paramètres et la fenêtre principal utilisée lorsque l'utilisateur a décidé de lancer le jeu. Ce package contient une vue pour tous les paquets de ressources du modèle et une vue de stratégie permmettant au joueur de savoir où en est la partie. Toutes les images sont chargées à l'aide du package ressources.
